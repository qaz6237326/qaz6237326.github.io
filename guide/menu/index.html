# Menu Attributes

参数 说明 数值类型 可选值 默认值 其他
data 数据 Array - -
nodeProps 自定义prop，可以自定义从哪个字段取值 Object { title, index, children, path, icon } { title, index, children, path, icon }
default-active 默认选中的值 String / Number - -
auto-jump 点击是否自动跳转 Boolean true / false true -
collapse 是否收起菜单 Boolean true / false false -
height 菜单高度 String - - -

# Menu Events

事件名 说明 回调参数
select 被点击的时候回调 index: 选中菜单项的 index, indexPath: 选中菜单项的 index path
click menuItem被点击的时候的回调 data：被点击的那一项的数据

# Menu 示例

用一个组件来示例

<template>
  <lx-menu
    :auto-jump="true"
    :collapse="collapse"
    :data="menuData"
    :default-active="defaultActive"
    height="100%"
    :menu-props="menuProps"
    :node-props="nodeProps"
    @click="selectNav"
  >
  </lx-menu>
</template>

<script>

export default {
  name: "nav-box",
  components: {},
  props: ['collapse'],
  data() {
    return {
      menuData: [],
      defaultActive: '',
      menuProps: {
        mode: 'vertical',
        backgroundColor: '#3A4651',
        textColor: '#fff'
      },
      nodeProps: {
        index: 'meta.menuId',
        children: 'children',
        path: 'path',
        hidden: 'meta.hidden',
        title: 'meta.title'
      }
    }
  },
  watch: {
    $route () {
      this.initActive()
    }
  },
  methods: {
    selectNav (navItem) {
      console.log('navItem', navItem)
    },
    initActive () {
      const { menuId, title } = this.$route.meta
      this.defaultActive = menuId
      document.title = title
    }
  },
  created() {
    this.initActive()
  }
}
</script>
